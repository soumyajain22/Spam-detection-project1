# 📩 Spam Message Detection using Machine Learning

## 📌 Overview
This project focuses on building a machine learning model that can automatically classify SMS messages as Spam or Not Spam (Ham). The system analyzes the content of a message and predicts whether it is unwanted or genuine.

## 🎯 Objective
The objective of this project is to detect spam messages using machine learning techniques and apply concepts learned in AI and ML.

## 🧠 Methodology
- Data preprocessing (cleaning text)
- Feature extraction using TF-IDF
- Model training using Naive Bayes
- Prediction and evaluation

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## 🧪 Example
Input: Win ₹5000 now!!!  
Output: Spam ❌

## 📊 Result
The model performs well in identifying spam messages based on learned patterns.
